This is the "((fuseWireComponent))" component.
We have two variables: a counter and a serverDateString:
  • Counter: (($counter))
  • Server time: (($serverDateString))
Reaction modes Explanation
CSR mode: this is client-side rendering, no server request is done, we only re-render in the client.
SERVER_WAIT mode: a request will be made to the server, and nothing will happen until the server responds.
SERVER mode: behaves like SERVER_WAIT, but will do a client-side re-rendering too.
SSR mode: this is server-side rendering (a.k.a. HTML-over-the-wire): all HTML content will be generated on the backend.
